Centrifugo User Presence Demo



How it works:

  1. Get OAuth2 token from POST /oauth2/token using Postman/curl
  2. Paste the access_token in the JWT Token field above
  3. Click "Connect" - it will call backend API to get Centrifugo token using your JWT
  4. Your connection will be established with the Centrifugo token
