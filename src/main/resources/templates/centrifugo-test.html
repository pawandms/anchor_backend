To get a JWT token:

  1. Use Postman or curl to get OAuth2 token from POST /oauth2/token
  2. Copy the access_token value
  3. Paste it in the JWT Token field above
  4. Click "Debug Token" to verify it has the correct "kid" header
  5. Click Connect
